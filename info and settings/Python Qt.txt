1. Установить интерпретатор с оф. сайта https://www.python.org/downloads/windows/
	устанавливать можно от пользователя
	при установке поставил галочку указать добавить path и поменял путь установки
	модуль PySide2 частично заработал с python-3.8.0-amd64
	
2. Установить PyCharm
	Можно в нем работать с PyQt6. Но зависимости подтягиваются отдельно каждый раз в каждый проект

3. Проверка
	проверить print("Hallo") в интерпретаторе
	проверить print("Hallo") в PyCharm
	
4. Установка библиотек
	открываем cmd и делем cd в [путь к интерпретатору]/Scripts
	выполняем команду pip install имя_библиотеки

	Библиотека PySide2 (используется стандартным пустым проектом в Qt) pip install PySide2
	так и не заработала, ругается на shiboken2

	Библиотека PyQt6 pip install pyqt6 и pip install pyqt-tools
	Пример программы (пустое окно): см файл Python QWidget.txt
